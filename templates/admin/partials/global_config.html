Guía: Obtener IDs de SharePoint

Sigue estos pasos en Graph Explorer:

1. Obtener Site ID

GET https://graph.microsoft.com/v1.0/sites/{hostname}:/sites/{sitename}

Ej: sites/miempresa.sharepoint.com:/sites/Comercial

Copia el valor del campo "id".

2. Obtener Drive ID (Document Library)

GET https://graph.microsoft.com/v1.0/sites/{SITE_ID}/drives

Busca el objeto con valor completo del campo id/drives (o tu librería).

Copia el valor del campo "id" (empieza con 'b!').

Ayuda: Configuración de KPIs

Fórmula de Cálculo

Score Final = (Score Base × Multiplicador)

Score Base = (%Compromiso × PesoComp) + (%Interno × PesoInt) + (%Volumen × PesoVol)
Multiplicador = 1.0 + (RatioLic × BonusLic) + (RatioAct × BonusAct) + Penalización

Ponderaciones (Pesos)

Determinan la importancia de cada métrica en el Score Base.

  • Peso Compromiso: Cumplimiento de fechas cliente (Default: 0.50).
  • Peso Interno: Cumplimiento SLA interno (Default: 0.35).
  • Peso Volumen: Cantidad entregada vs Max (Default: 0.15).

Factores y Multiplicadores

  • Ratio Licitaciones (Umbral): Si un usuario supera este % de licitaciones, se clasifica como "Alta Complejidad". (No afecta fórmula directa, solo categoría).
  • Mult. Licitaciones: Bonus por complejidad de licitaciones (ej: 0.20 suma 20% al multiplicador por cada punto de ratio).
  • Penalización Retrabajos: Valor Negativo. Resta puntos por retrabajos (ej: -0.15 resta 15%).

Semáforo (Umbrales)

Verde: Score >= 90
Ámbar: 85 <= Score < 90
Rojo: Score < 85
