Integración SharePoint (Microsoft Graph)

Si se define, todos los adjuntos se guardarán dentro de esta carpeta.

Recomendado: 500 MB (Máx soportado por red: 5 GB)

Guía: Obtener IDs de SharePoint

Sigue estos pasos en Graph Explorer:

1. Obtener Site ID

GET https://graph.microsoft.com/v1.0/sites/{hostname}:/sites/{sitename}

Ej: sites/miempresa.sharepoint.com:/sites/Comercial

Copia el valor del campo "id".

2. Obtener Drive ID (Document Library)

GET https://graph.microsoft.com/v1.0/sites/{SITE_ID}/drives

Busca el objeto con "name": "Documents" (o tu librería).

Copia el valor del campo "id" (empieza con 'b!').
